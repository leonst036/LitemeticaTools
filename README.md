# LitemeticaTools

Ein GUI-basiertes Tool-Set in Python zur Arbeit mit Minecraft Schematics.

## Funktionen

* **Konverter:** Konvertierung zwischen dem Litematica-Format (`.litematica`) und dem Sponge Schematic-Format (`.schem`).
* **Materialliste:** Es lassen sich Materiallisten aus Litematica- und Schematica-Dateien generieren. 
* **Übersichtliche Darstellung:** Die Mengen der Blöcke für die Materialliste werden automatisch in Shulkerkisten (SB), Stacks (Stk) und einzelne Items umgerechnet (z. B. `1 SB + 2 Stk + 3`).
